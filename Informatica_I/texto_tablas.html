¿Qué es el Formato de Texto?

El formato de texto se refiere a la presentación y estilo del texto en un documento, como el tipo de fuente, tamaño, color y alineación. Es fundamental para mejorar la legibilidad y la estética de un documento.

Herramientas de Formato de Texto

  • Fuente: Permite cambiar el tipo de letra (Times New Roman, Arial, etc.).
  • Tamaño de Fuente: Ajusta el tamaño del texto.
  • Estilo de Fuente: Incluye negrita, cursiva y subrayado para enfatizar palabras o frases.
  • Alineación: Posiciona el texto a la izquierda, derecha, centro o justificado.

¿Qué son las Tablas?

Una tabla es una estructura que organiza la información en filas y columnas. Es una herramienta esencial en procesadores de texto para presentar datos de manera ordenada y clara.

Componentes de una Tabla

  • Fila: Un conjunto horizontal de celdas.
  • Columna: Un conjunto vertical de celdas.
  • Celda: El recuadro individual que se forma en la intersección de una fila y una columna.
  • Encabezado: La primera fila o columna que describe el contenido del resto de la tabla.
