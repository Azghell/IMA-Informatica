Organizando y Midiendo la Información Digital

Comprender cómo se almacena, organiza y se accede a la información digital es esencial en la informática. Desde la foto más pequeña hasta un sistema operativo completo, todo tiene un tamaño y un formato específico.

  • Exploraremos las medidas de almacenamiento (bits, bytes, etc.).
  • Entenderemos los formatos de archivos más comunes y su propósito.
  • Aprenderemos a navegar y gestionar archivos en nuestros sistemas.

Medidas de Almacenamiento: El Lenguaje Binario

La información en las computadoras se almacena utilizando el sistema binario, compuesto por bits.

  • Bit (Binary Digit): Es la unidad más pequeña de información, que puede ser un 0 o un 1. Es el "átomo" de los datos digitales.
  • Byte: Un grupo de 8 bits. Un byte es la unidad básica para representar un carácter (una letra, un número, un símbolo).

Jerarquía de Medidas de Almacenamiento

Las unidades de almacenamiento crecen exponencialmente, permitiéndonos medir desde el tamaño de un texto hasta la capacidad de un disco duro.

Kilobyte (KB)

≈ 1,000 Bytes (1024 Bytes). Un pequeño documento de texto.

Megabyte (MB)

≈ 1,000 KB (1,048,576 Bytes). Una canción MP3, una imagen de alta calidad.

Gigabyte (GB)

≈ 1,000 MB (1,073,741,824 Bytes). Una película, un software grande.

Terabyte (TB)

≈ 1,000 GB. Cientos de películas en HD, gran capacidad de disco duro.

Petabyte (PB)

≈ 1,000 TB. Grandes centros de datos, almacenamiento a escala masiva.

Exabyte (EB)

≈ 1,000 PB. Escala de datos global de internet.

Formatos de Archivos: La Estructura de los Datos

El formato de archivo define cómo se organizan y almacenan los datos dentro de un archivo. Esta estructura permite a los programas informáticos interpretar y procesar correctamente la información.

  • Cada formato tiene una extensión de archivo (.txt, .jpg, .mp4) que indica su tipo.
  • La elección del formato afecta la compatibilidad, el tamaño y la calidad del archivo.

Tipos Comunes de Archivos: Documentos

Los archivos de documentos son fundamentales para la creación y compartición de información escrita.

Texto Plano (.txt)

Contiene solo caracteres de texto sin formato (negrita, cursiva, etc.). Ligero y universal.

Documento de Word (.doc, .docx)

Archivos de procesadores de texto con formato rico, imágenes y objetos. .docx es el formato moderno de Microsoft Word.

Documento PDF (.pdf)

Portable Document Format. Mantiene el formato original en cualquier dispositivo. Ideal para compartir.

Hoja de Cálculo (.xls, .xlsx)

Archivos para datos tabulares, fórmulas y gráficos. Usados en Microsoft Excel.

Presentación (.ppt, .pptx)

Archivos de diapositivas con texto, imágenes y transiciones. Usados en Microsoft PowerPoint.

Tipos Comunes de Archivos: Multimedia y Comprimidos

Más allá de los documentos, existen formatos especializados para medios y para la gestión de espacio.

Imágenes (.jpg, .png, .gif)

JPEG (fotos), PNG (transparencia, logos), GIF (animaciones cortas).

Audio (.mp3, .wav)

MP3 (comprimido, popular), WAV (sin comprimir, alta calidad).

Video (.mp4, .avi, .mov)

MP4 (web, móvil), AVI (calidad, tamaño grande), MOV (Apple QuickTime).

Comprimidos (.zip, .rar)

Contenedores de uno o más archivos para reducir su tamaño o agruparlos.

Ejecutables (.exe, .app)

Programas o aplicaciones que se pueden ejecutar directamente en un sistema operativo.

Navegación entre Archivos: El Sistema de Organización

El sistema de archivos es la estructura que un sistema operativo utiliza para controlar cómo se almacenan y recuperan los datos. Organiza los archivos en una estructura jerárquica de directorios (carpetas).

  • Carpetas (Directorios): Contenedores para organizar archivos y otras carpetas.
  • Carpeta Raíz: La carpeta principal de una unidad de almacenamiento (ej., C:\ en Windows, / en Linux/macOS).
  • Rutas de Archivos: La dirección única de un archivo o carpeta en el sistema de archivos.
    • Ruta Absoluta: Desde la carpeta raíz (ej., C:\Usuarios\Documentos\archivo.txt).
    • Ruta Relativa: Desde la ubicación actual (ej., ..\otra_carpeta\imagen.jpg).

Gestión de Archivos: Uso del Explorador

El Explorador de Archivos (o Finder en macOS) es la herramienta gráfica principal para interactuar con el sistema de archivos.

  • Crear: Click derecho -> Nuevo -> Carpeta/Documento.
  • Eliminar: Seleccionar y presionar Supr, o arrastrar a la Papelera de Reciclaje.
  • Renombrar: Click derecho -> Cambiar nombre, o F2.
  • Mover: Cortar (Ctrl+X) y Pegar (Ctrl+V) en otra ubicación.
  • Copiar: Copiar (Ctrl+C) y Pegar (Ctrl+V) en otra ubicación.
  • Buscar: Usar la barra de búsqueda para encontrar archivos o carpetas por nombre, fecha, tipo.
