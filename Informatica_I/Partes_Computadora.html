Partes y Tipos de Computadora

Hardware vs. Software
_
▢
x

Hardware vs. Software

Dos componentes esenciales para el funcionamiento de una computadora.

  • Hardware: Componentes físicos y tangibles.
  • Software: Programas e instrucciones intangibles.
La CPU: El Cerebro
_
▢
x

La CPU: El Cerebro

La Unidad Central de Procesamiento (CPU) ejecuta instrucciones, realiza cálculos y procesa datos.

  • Unidad Aritmético-Lógica (ALU): Operaciones matemáticas y lógicas.
  • Unidad de Control (UC): Dirige el flujo de datos.
  • Registros: Almacenamiento temporal de alta velocidad.
Memoria RAM y ROM
_
▢
x

Memoria RAM y ROM

Fundamental para el almacenamiento de datos.

  • RAM (Acceso Aleatorio): Memoria principal, volátil, almacena datos en uso.
  • ROM (Solo Lectura): Memoria no volátil, almacena instrucciones de arranque (BIOS/UEFI).
Placa Madre (Motherboard)
_
▢
x

Placa Madre (Motherboard)

La tarjeta principal que conecta todos los componentes de la computadora.

  • Sistema nervioso central del equipo.
  • Permite la comunicación entre CPU, RAM, tarjetas y puertos.
Unidades de Almacenamiento
_
▢
x

Unidades de Almacenamiento

Almacenamiento de datos permanente.

  • HDD (Disco Duro): Almacenamiento magnético, más lento, económico.
  • SSD (Estado Sólido): Basado en memoria flash, más rápido y duradero.
Fuente de Poder y Tarjetas de Expansión
_
▢
x

Fuente de Poder y Tarjetas de Expansión

  • Fuente de Poder: Convierte la corriente eléctrica para los componentes.
  • Tarjetas de Expansión: Añaden funcionalidades específicas (Gráfica, Sonido, Red).

Tipos de Computadoras

Escritorio y Portátiles

  • De Escritorio: Fijas, potentes, personalizables.
  • Portátiles: Compactas, movilidad, todo en uno.
Servidores y Tabletas
_
▢
x

Servidores y Tabletas

  • Servidores: Potentes, alta disponibilidad, gestionan recursos de red.
  • Tabletas: Portátiles, pantalla táctil, consumo de medios.
Smartphones y Supercomputadoras
_
▢
x

Smartphones y Supercomputadoras

  • Smartphones: Móviles, comunicación celular, aplicaciones.
  • Supercomputadoras: Extremadamente potentes, para investigación avanzada.
Sistemas Integrados (Embedded)
_
▢
x

Sistemas Integrados (Embedded)

Sistemas computacionales especializados para funciones dedicadas dentro de un sistema más grande.

  • Ejemplos: Electrodomésticos inteligentes, vehículos, IoT.
