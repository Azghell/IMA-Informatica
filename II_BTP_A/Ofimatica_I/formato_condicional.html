¿Qué es el Formato Condicional?

El formato condicional es una herramienta que aplica automáticamente un estilo visual (como un color, un ícono o una barra de datos) a una celda o un rango de celdas, basándose en una regla o condición que tú defines.

Es una forma de hacer que tus datos "hablen" por sí mismos, permitiéndote identificar patrones, tendencias o problemas de un vistazo.

Usos Comunes

Puedes usar el formato condicional para:

  • Resaltar datos importantes: Marcar ventas por encima del objetivo, notas de estudiantes aprobadas, o fechas de vencimiento cercanas.
  • Identificar anomalías: Destacar valores duplicados o errores en un conjunto de datos.
  • Visualizar tendencias: Usar barras de datos, escalas de color o conjuntos de íconos para mostrar el progreso de un vistazo.
  • Analizar el rendimiento: Ver fácilmente el desempeño de diferentes productos o empleados.

Fórmulas para Establecer Reglas

Las reglas de formato condicional pueden ser simples o complejas. Puedes usar fórmulas para crear condiciones personalizadas.

Ejemplo de una regla simple:

El valor de la celda es mayor que 50

Ejemplo usando una fórmula:

=C2="Aprobado" (Resalta celdas que contienen la palabra "Aprobado")

`=Y(C2>90, C2<100)` (Resalta valores entre 90 y 100)

Pasos para Aplicar Formato Condicional

  • Selecciona tus datos: Haz clic y arrastra para seleccionar el rango de celdas al que deseas aplicar el formato.
  • Abre la herramienta: Busca la opción "Formato condicional" en la barra de herramientas. En Excel, está en la pestaña `Inicio`.
  • Elige una regla: Selecciona una de las reglas predefinidas (ej. `Es mayor que...`, `Texto que contiene...`) o elige la opción `Nueva regla...` para usar una fórmula.
  • Configura el formato: Elige el color de fondo, el color de la fuente, etc., que se aplicará cuando se cumpla la condición.
  • Aplica la regla: Confirma los ajustes y el formato se aplicará automáticamente a tus datos.

Tipos de Reglas y Visualización

Además de resaltar celdas, el formato condicional ofrece herramientas visuales para un análisis más rápido.

Barras de Datos

Una barra horizontal en la celda que muestra su valor en relación con otros.

Escalas de Color

Un degradado de color que cambia según el valor de la celda en un rango.

Conjuntos de Íconos

Agrega un ícono a la celda para representar su valor (ej. flechas, semáforos).

Cómo Aplicar Visualizaciones

Sigue estos pasos para aplicar barras de datos, escalas de color o conjuntos de íconos.

  • Selecciona un rango de datos que contenga solo números.
  • Ve a la herramienta "Formato Condicional" y selecciona la opción que deseas:
    • Para Barras de Datos: Elige un color de la lista.
    • Para Escalas de Color: Elige un degradado de color.
    • Para Conjuntos de Íconos: Elige el conjunto de íconos que prefieras (flechas, círculos, semáforos, etc.).
  • La aplicación de hoja de cálculo hará el resto automáticamente, basándose en la distribución de tus valores.

Gestión de Reglas: Prioridad

Cuando tienes varias reglas aplicadas a la misma celda, su orden es crucial.

La regla que se ejecuta primero es la que manda.

  • Las reglas se aplican en el orden en que aparecen en la lista del "Administrador de reglas".
  • Puedes cambiar la prioridad de una regla moviéndola hacia arriba o hacia abajo en la lista.
  • Si una celda cumple con la primera regla, puedes marcar la opción "Detener si es verdad" para evitar que se apliquen otras reglas posteriores.

Gestión de Reglas: Edición y Eliminación

La clave para mantener tu hoja de cálculo ordenada es saber cómo modificar o eliminar reglas.

  • Para editar una regla, simplemente selecciónala en el "Administrador de reglas" y haz clic en "Editar regla".
  • Para eliminar una regla que ya no necesitas, selecciónala y haz clic en "Eliminar regla".
  • Esto te permite ajustar tus visualizaciones sin tener que empezar de cero cada vez.

Actividad Práctica

Instrucciones:

Crea una tabla en una hoja de cálculo con la información de al menos 10 productos y sus precios. Aplica el formato condicional para:

  • Resaltar en verde los productos con un precio menor a $10.
  • Resaltar en rojo los productos con un precio mayor a $100.
  • Usar una escala de color para mostrar el rango de precios de todos los productos.

Preguntas de Repaso

Resumen

El formato condicional es una herramienta esencial para el análisis de datos. Te ayuda a detectar tendencias, a identificar valores importantes y a tomar decisiones más informadas, simplemente aplicando estilos visuales a tus datos.

Tarea

Instrucciones:

En una hoja de cálculo, crea una fórmula de formato condicional para resolver cada uno de los siguientes escenarios:

  • Producto agotado: Resalta toda la fila de un producto si la celda en la columna "Estado" dice "Agotado".
  • Ventas superiores al promedio: Resalta en verde todas las celdas de la columna "Ventas" que sean superiores al promedio de todas las ventas.
  • Fechas de vencimiento: Resalta en rojo las celdas en la columna "Fecha de vencimiento" si la fecha ya pasó.
  • Top 3: Resalta los 3 valores más altos en una lista de puntuaciones.
  • Celdas no vacías: Resalta las celdas en un rango si no están vacías.
