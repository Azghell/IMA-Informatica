Instalación de Software y Virtualización

Informática Aplicada I

Undécimo A

Docente: Pablo Antonio Peña

San Pedro Sula

03 de agosto del 2025

¿Qué es la Instalación de Software?

Proceso de preparar un programa para su ejecución en una computadora.

  • Copiar archivos, configurar ajustes del sistema.
  • Crear accesos directos y registrar componentes.

Tipos de Instaladores (GUI)

Los más comunes para usuarios, con asistentes visuales.

Windows

  • .exe (ejecutables autoextraíbles)
  • .msi (Microsoft Installer)

macOS

  • .dmg (Imágenes de disco)
  • .pkg (Paquetes de instalación)

Instalación por Línea de Comandos (CLI)

Uso de la terminal para mayor control y automatización.

Gestores de Paquetes:

  • apt (Debian/Ubuntu): `sudo apt install nombre_paquete`
  • yum / dnf (Red Hat/Fedora)
  • pip (Python): `pip install nombre_modulo`

Ventajas: Automatización, resolución de dependencias.

¿Qué es la Virtualización?

Tecnología para crear versiones virtuales de recursos informáticos.

Máquina Virtual (VM):

  • Emulación de un sistema informático completo.
  • Ejecuta un SO invitado sobre un SO anfitrión.
  • Controlada por un Hypervisor.

Ventajas de la Virtualización

Beneficios clave de usar máquinas virtuales.

  • Aislamiento: Entornos separados y seguros.
  • Consolidación: Múltiples SO en una sola máquina física.
  • Portabilidad: Mover VMs fácilmente entre hosts.
  • Desarrollo y Pruebas: Entornos seguros para experimentar.
  • Seguridad: Ejecutar software de riesgo sin afectar el sistema principal.

Tipos de Hypervisores

El software que gestiona las máquinas virtuales.

Tipo 1 (Bare-Metal)

  • Se instala directamente sobre el hardware.
  • Alto rendimiento, seguridad.
  • Ejemplos: VMware ESXi, Microsoft Hyper-V.

Tipo 2 (Hosted)

  • Se instala como una aplicación sobre un SO anfitrión.
  • Más fácil de usar para usuarios domésticos.
  • Ejemplos: Oracle VirtualBox, VMware Workstation.

Software de Virtualización Popular

Herramientas ampliamente utilizadas para crear y gestionar VMs.

Oracle VirtualBox

Gratuito, código abierto (Tipo 2)

VMware Workstation / Fusion

Comercial, profesional (Tipo 2)

Microsoft Hyper-V

Integrado en Windows Pro/Enterprise (Tipo 1)

¿Sabías que...?

  • La virtualización se originó en IBM en la década de 1960.
  • El término "sandbox" (entorno aislado) está relacionado con la virtualización.
  • La virtualización es la base de la "Computación en la Nube" (Cloud Computing).
  • Puedes instalar casi cualquier SO en una VM, ideal para pruebas.

Asignación de Tarea

Para entregar el 10 de agosto de 2025:

Redacta un ensayo sobre las precauciones y buenas costumbres que se deben tener al momento de instalar programas en una computadora. El ensayo debe tener una extensión de:

  • Página y media (1.5 páginas) si usas un cuaderno espiral grande.
  • Dos páginas (2 páginas) si usas un cuaderno pequeño.
Asegúrate de incluir aspectos como: verificar la fuente del software, leer los términos y condiciones, evitar barras de herramientas no deseadas, el uso de antivirus, y la importancia de los permisos de instalación.
