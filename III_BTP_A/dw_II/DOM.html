Dominio del DOM

El Document Object Model (DOM) es una interfaz de programación para documentos HTML y XML.

  • Representa la estructura del documento como un árbol de nodos.
  • Permite que programas (como JavaScript) accedan y manipulen el contenido, estructura y estilo de una página web.

Estructura del DOM (Árbol)

El DOM organiza los elementos de una página web en una estructura jerárquica, similar a un árbol.

  • Cada parte del documento (elementos, atributos, texto) es un nodo.
  • Los nodos tienen relaciones: padres, hijos, hermanos.

Tipos de Nodos Principales

El DOM se compone de diferentes tipos de nodos:

Nodo Documento

Representa todo el documento HTML (el objeto document). Es la raíz del árbol.

Nodo Elemento

Representa una etiqueta HTML (ej. <div>, <p>, <img>). Puede tener hijos y atributos.

Nodo Texto

Representa el contenido de texto dentro de un elemento (ej. "Hola mundo" dentro de <p>).

Nodo Atributo

Representa un atributo de una etiqueta HTML (ej. id="miId", class="clase", src="imagen.jpg").

Selección de Elementos del DOM

Antes de poder manipular cualquier parte de una página web con JavaScript, primero debemos seleccionar los elementos específicos con los que queremos interactuar.

Existen varios métodos para hacer esto, y cada uno es útil en diferentes situaciones.

El Objeto Global document

En el contexto del DOM, document es el objeto global que representa toda la página web cargada en el navegador.

  • Es la puerta de entrada a todo el contenido de la página HTML.
  • Contiene métodos y propiedades para acceder, crear y modificar elementos, manejar eventos y mucho más.
  • Piensa en document como el "cerebro" o el "controlador principal" de tu página web.

// Accediendo al título de la página a través del objeto document
const tituloPagina = document.title;
console.log(tituloPagina); // Muestra el título de la etiqueta <title>

document.getElementById('idElemento')

Este método es el más común para seleccionar un único elemento en tu HTML, siempre y cuando tenga un atributo id único.

  • getElementById: Es el método que busca un elemento por el valor de su atributo id.
  • El id debe ser único en todo el documento HTML.
  • Devuelve el elemento encontrado o null si no lo encuentra.

<!-- archivo HTML -->
<div id="miContenedor">
  <p>Este es un párrafo dentro del contenedor.</p>
</div>

<!-- JavaScript -->
<script>
  const contenedor = document.getElementById('miContenedor');
  console.log(contenedor.textContent); // "Este es un párrafo dentro del contenedor."
</script>

document.querySelector('selectorCSS')

Este método es muy versátil, ya que te permite seleccionar el primer elemento que coincida con un selector CSS específico.

  • querySelector: Busca un elemento usando cualquier selector CSS (clases, IDs, etiquetas, atributos, etc.).
  • Si hay múltiples coincidencias, solo devuelve la primera.
  • Devuelve el elemento o "null" si no lo encuentra.

<!-- HTML de ejemplo -->
<div class="caja">Caja Uno</div>
<div class="caja">Caja Dos</div>
<p id="mensaje">Hola mundo.</p>

<!-- JavaScript -->
<script>
  const primeraCaja = document.querySelector('.caja'); // Selecciona la primera div con clase "caja"
  console.log(primeraCaja.textContent); // "Caja Uno"
  const miMensaje = document.querySelector('#mensaje'); // Selecciona por ID
</script>

document.querySelectorAll('selectorCSS')

Similar a querySelector, pero este método devuelve una NodeList (una colección de nodos) de todos los elementos que coinciden con el selector CSS.

  • querySelectorAll: Busca todos los elementos que cumplen con el selector.
  • La NodeList devuelta es similar a un array, permitiendo iterar sobre los elementos (ej. con forEach).
  • Si no encuentra coincidencias, devuelve una NodeList vacía.

<!-- HTML de ejemplo -->
<li class="item-lista">Manzana</li>
<li class="item-lista">Pera</li>
<li class="item-lista">Naranja</li>

<!-- JavaScript -->
<script>
  const items = document.querySelectorAll('.item-lista');
  console.log(items.length); // 3
  items.forEach(item => {
    console.log(item.textContent); // "Manzana", "Pera", "Naranja"
  });
</script>

document.getElementsByClassName('nombreClase')

Este método selecciona todos los elementos que tienen una clase CSS específica.

  • getElementsByClassName: Busca elementos por su atributo "class".
  • Devuelve un HTMLCollection, que es similar a un array, pero no tiene todos los métodos de un array (ej. no forEach directo, se necesita convertir o usar un bucle for).

<!-- HTML de ejemplo -->
<p class="texto-resaltado">Importante 1</p>
<span class="texto-resaltado">Importante 2</span>

<!-- JavaScript -->
<script>
  const resaltados = document.getElementsByClassName('texto-resaltado');
  console.log(resaltados.length); // 2
  // Para usar forEach, convertir a Array: Array.from(resaltados).forEach(...)
</script>

document.getElementsByTagName('nombreEtiqueta')

Este método te permite seleccionar todos los elementos que tienen una etiqueta HTML específica (ej. <p>, <div>, <a>).

  • getElementsByTagName: Busca elementos por su nombre de etiqueta HTML.
  • También devuelve un HTMLCollection.

<!-- HTML de ejemplo -->
<ul>
  <li>Elemento de lista 1</li>
  <li>Elemento de lista 2</li>
</ul>

<!-- JavaScript -->
<script>
  const listItems = document.getElementsByTagName('li');
  console.log(listItems.length); // 2
</script>

Modificación de Contenido del DOM

Una vez que hemos seleccionado un elemento, una de las tareas más comunes es cambiar su contenido, ya sea texto simple o incluso añadir más HTML.

Exploraremos dos propiedades clave para lograr esto.

elemento.textContent

La propiedad textContent te permite obtener o establecer el contenido de texto plano de un elemento y todos sus descendientes.

  • Solo maneja texto; cualquier etiqueta HTML dentro se interpretará como texto literal.
  • Es útil para actualizar texto simple de forma segura, evitando inyecciones de HTML maliciosas.

<!-- HTML de ejemplo -->
<p id="saludo">Hola, <strong>usuario</strong>.</p>

<!-- JavaScript -->
<script>
  const parrafoSaludo = document.getElementById('saludo');
  console.log(parrafoSaludo.textContent); // "Hola, usuario."
  parrafoSaludo.textContent = '¡Bienvenido al DOM!'; // Cambia el texto
</script>

elemento.innerHTML

La propiedad innerHTML te permite obtener o establecer el contenido HTML completo (incluyendo etiquetas) de un elemento.

  • Puedes insertar HTML válido, y el navegador lo renderizará.
  • Cuidado: Puede ser vulnerable a ataques de inyección de código (XSS) si se usa con contenido no confiable.

<!-- HTML de ejemplo -->
<div id="areaContenido">
  <p>Contenido original.</p>
</div>

<!-- JavaScript -->
<script>
  const area = document.getElementById('areaContenido');
  area.innerHTML = '<h3>Nuevo Título</h3><p>Texto <strong>dinámico</strong>.</p>';
</script>

Modificación de Atributos y Estilos

Más allá del contenido, el DOM nos permite controlar las propiedades (atributos) y la apariencia (estilos) de los elementos HTML.

Esto es crucial para la interactividad y el diseño dinámico.

setAttribute(), getAttribute(), removeAttribute()

Estos métodos permiten manipular los atributos de un elemento HTML (ej. src de una imagen, href de un enlace, data-* personalizados).

  • setAttribute(nombre, valor): Establece o cambia el valor de un atributo.
  • getAttribute(nombre): Obtiene el valor actual de un atributo.
  • removeAttribute(nombre): Elimina un atributo de un elemento.

<!-- HTML de ejemplo -->
<img id="miImagen" src="original.jpg" alt="Imagen original">
<a id="miEnlace" href="#">Ir a Google</a>

<!-- JavaScript -->
<script>
  const imagen = document.getElementById('miImagen');
  imagen.setAttribute('src', 'nueva.jpg'); // Cambia la fuente
  console.log(imagen.getAttribute('alt')); // "Imagen original"
  imagen.removeAttribute('alt'); // Elimina el atributo alt
  document.getElementById('miEnlace').setAttribute('href', 'https://www.google.com');
</script>

elemento.style.propiedadCSS

La propiedad style de un elemento te permite acceder y modificar directamente los estilos CSS en línea de ese elemento.

  • Cada propiedad CSS se convierte en una propiedad JavaScript (ej. background-color es backgroundColor).
  • Ideal para cambios de estilo rápidos y específicos, pero para estilos complejos o múltiples, es mejor usar clases CSS.

<!-- HTML de ejemplo -->
<div id="cajaEstilo">Esta caja cambiará de estilo.</div>

<!-- JavaScript -->
<script>
  const caja = document.getElementById('cajaEstilo');
  caja.style.backgroundColor = '#FFD700'; // Color oro
  caja.style.border = '2px solid #DAA520';
  caja.style.padding = '10px';
</script>

elemento.classList (add, remove, toggle)

La propiedad classList es la forma preferida de manipular las clases CSS de un elemento, lo que permite aplicar y quitar grupos de estilos de manera eficiente.


  • add('clase'): Añade una o más clases.
  • remove('clase'): Elimina una o más clases.
  • toggle('clase'): Añade la clase si no existe, la elimina si sí existe.
  • contains('clase'): Verifica si un elemento tiene una clase.


<!-- HTML de ejemplo -->
<button id="botonAccion" class="btn">Activar/Desactivar</button>
<style>
  .btn { padding: 10px; background-color: gray; }
  .activo { background-color: green; color: white; }
</style>

<!-- JavaScript -->
<script>
  const boton = document.getElementById('botonAccion');
  boton.addEventListener('click', () => {
    boton.classList.toggle('activo'); // Alterna la clase 'activo'
    console.log(boton.classList.contains('activo'));
  });
</script>

Creación y Eliminación de Elementos

El DOM no solo permite modificar elementos existentes, sino también crear nuevos desde cero o eliminar los que ya no son necesarios.

Esto es fundamental para construir interfaces de usuario dinámicas y reactivas.

document.createElement('etiqueta')

Este método crea un nuevo nodo de elemento HTML del tipo especificado (ej. <div>, <p>, <img>).

  • El elemento se crea en memoria, pero no se añade automáticamente a la página.
  • Necesitas usar métodos como appendChild() para insertarlo en el DOM visible.

// Crear un nuevo párrafo
const nuevoParrafo = document.createElement('p');

// Asignar contenido de texto
nuevoParrafo.textContent = 'Este es un párrafo creado dinámicamente.';

// Asignar una clase CSS (opcional)
nuevoParrafo.classList.add('mensaje-dinamico');

// En este punto, 'nuevoParrafo' existe, pero aún no está en la página.

padre.appendChild(hijo)

Este método añade un nodo (elemento, texto, etc.) como el último hijo de un elemento padre especificado.

  • Es la forma más común de insertar elementos en el DOM.
  • Si el nodo hijo ya existe en el DOM, appendChild lo moverá a la nueva posición.

<!-- HTML de ejemplo -->
<div id="listaTareas">
  <p>Tareas pendientes:</p>
  <ul id="miLista">
    <li>Comprar leche</li>
  </ul>
</div>

<!-- JavaScript -->
<script>
  const lista = document.getElementById('miLista');
  const nuevoItem = document.createElement('li');
  nuevoItem.textContent = 'Estudiar DOM';
  lista.appendChild(nuevoItem); // Añade "Estudiar DOM" al final de la lista
</script>

padre.removeChild(hijo)

Este método elimina un nodo hijo específico del DOM. Para que funcione, debes llamar al método desde el elemento padre del nodo que quieres eliminar.

  • Necesitas una referencia tanto al padre como al hijo.
  • Es importante verificar si el hijo existe antes de intentar eliminarlo para evitar errores.

<!-- HTML de ejemplo -->
<div id="contenedorPadre">
  <p id="parrafoAborrar">Este párrafo desaparecerá.</p>
  <span>Este span se quedará.</span>
</div>

<!-- JavaScript -->
<script>
  const padre = document.getElementById('contenedorPadre');
  const hijoAEliminar = document.getElementById('parrafoAborrar');

  if (hijoAEliminar) { // Verificar que el elemento existe
    padre.removeChild(hijoAEliminar); // Elimina el párrafo
  }
</script>

Eventos del DOM

Los eventos son la forma en que JavaScript "escucha" y reacciona a las interacciones del usuario o a los cambios en el navegador.

Permiten que tu página web sea interactiva, respondiendo a clics, pulsaciones de teclas, envíos de formularios y más.

elemento.addEventListener('evento', funcion)

Este método es la forma estándar y recomendada de registrar un "escuchador de eventos" en un elemento.


  • 'evento': El tipo de evento a escuchar (ej. 'click', 'mouseover', 'keydown').
  • funcion: La función que se ejecutará cuando el evento ocurra.
  • Puedes añadir múltiples escuchadores para el mismo evento en el mismo elemento.


<!-- HTML de ejemplo -->
<button id="botonInteractivo">Haz Clic Aquí</button>
<p id="mensajeInteractivo">Esperando interacción...</p>

<!-- JavaScript -->
<script>
  const boton = document.getElementById('botonInteractivo');
  const mensaje = document.getElementById('mensajeInteractivo');

  boton.addEventListener('click', function() {
    mensaje.textContent = '¡Has hecho clic en el botón!';
    boton.style.backgroundColor = '#34D399'; // Verde
  });

  boton.addEventListener('mouseover', () => {
    boton.style.transform = 'scale(1.05)';
  });

  boton.addEventListener('mouseout', () => {
    boton.style.transform = 'scale(1)';
  });
</script>
