Fundamentos de JavaScript

¿Qué es JavaScript?

JavaScript (JS) es un lenguaje de programación ligero e interpretado, fundamental para el desarrollo web moderno.

  • Permite crear contenido dinámico e interactivo en sitios web.
  • Se ejecuta principalmente en el lado del cliente (navegador del usuario).
  • Es compatible con HTML y CSS para construir experiencias web completas.
  • También se usa en el lado del servidor (Node.js), aplicaciones móviles y de escritorio.

Variables en JavaScript

Las variables son contenedores para almacenar datos. JS tiene tres palabras clave para declararlas:

  • var: Declaración antigua. Alcance de función o global. Puede ser redeclarada y reasignada.
  • let: Introducido en ES6. Alcance de bloque. Puede ser reasignada, pero no redeclarada en el mismo alcance.
  • const: Introducido en ES6. Alcance de bloque. No puede ser reasignada ni redeclarada. Debe inicializarse.

Ejemplo: Declaración de Variables

Sintaxis del Código:

var nombre = "Ana"; let edad = 30; const PI = 3.14159; console.log("Nombre: " + nombre); console.log("Edad inicial: " + edad); edad = 31; // Válido con let, reasigna el valor console.log("Nueva edad: " + edad); // PI = 3.14; // Error: No se puede reasignar const

Tipos de Datos Fundamentales

JavaScript maneja varios tipos de datos para representar información:

  • String: Texto (ej. "Hola mundo").
  • Number: Números enteros o decimales (ej. 10, 3.14).
  • Boolean: Valores lógicos true o false.
  • Array: Colección ordenada de elementos (ej. [1, 2, 3]).
  • Object: Colección de pares clave-valor (ej. {nombre: "Juan", edad: 25}).

Ejemplo: Tipos de Datos

Sintaxis del Código:

let mensaje = "Hola"; // String let cantidad = 100; // Number let activo = true; // Boolean let colores = ["rojo", "verde"]; // Array let usuario = {id: 1, nombre: "Maria"}; // Object console.log("Tipo de mensaje: " + typeof mensaje); console.log("Tipo de cantidad: " + typeof cantidad); console.log("Tipo de activo: " + typeof activo); console.log("Tipo de colores: " + typeof colores); // Arrays son de tipo 'object' console.log("Tipo de usuario: " + typeof usuario);

Operadores en JavaScript

Los operadores se usan para realizar operaciones sobre valores (operandos).

  • Aritméticos: +, -, *, /, % (módulo).
  • Asignación: =, +=, -=, *=, /=.
  • Comparación: == (igualdad de valor), === (igualdad estricta), !=, !==, <, >, <=, >=.
  • Lógicos: && (AND), || (OR), ! (NOT).

Ejemplo: Operadores

Sintaxis del Código:

let x = 5; let y = 3; let resultado = x + y; // Cambia '+' por '-', '*', '/' o '%' para otras operaciones. console.log("Resultado de la operación: " + resultado); x += 2; // x es ahora 7 (operador de asignación) console.log("Nuevo valor de x (después de x += 2): " + x); let esMayor = x > y; // operador de comparación console.log("¿Es x mayor que y?: " + esMayor); let condicion = (x > 0 && y < 10); // operador lógico console.log("Condición lógica (x > 0 && y < 10): " + condicion);

Estructuras de Control: Condicionales

Permiten ejecutar código basado en ciertas condiciones.

  • if / else if / else: Para ejecutar bloques de código condicionalmente.
  • switch: Para evaluar una expresión contra múltiples casos.

Ejemplo: Condicional if/else

Sintaxis del Código:

let hora = 14; // Prueba con diferentes horas, ej. 9, 14, 20 if (hora < 12) { console.log("Buenos días"); } else if (hora < 18) { console.log("Buenas tardes"); } else { console.log("Buenas noches"); }

Estructuras de Control: Bucles

Permiten repetir un bloque de código múltiples veces.

  • for: Para bucles con un número conocido de iteraciones.
  • while: Para bucles que continúan mientras una condición sea verdadera.

Ejemplo: Bucle for

Sintaxis del Código:

for (let i = 0; i < 5; i++) { console.log("Iteración " + i); }

Funciones en JavaScript

Las funciones son bloques de código reutilizables que realizan una tarea específica.

  • Declaración de función: Definidas con la palabra clave function.
  • Expresión de función: Asignadas a una variable.
  • Funciones flecha (Arrow Functions): Sintaxis más concisa (ES6).

Ejemplo: Declaración de Función

Sintaxis del Código:

// Declaración de función function saludar(nombre) { return "Hola, " + nombre + "!"; } console.log(saludar("Pedro")); // Llama a la función console.log(saludar("María"));

Cómo insertar JavaScript en HTML

Hay tres formas principales de integrar JavaScript en tus páginas web:

1. JavaScript Inline

Directamente en las etiquetas HTML usando atributos de evento.

Sintaxis del Código:

<!-- HTML --> <button onclick="alert('¡Hola desde JS Inline!')">Haz clic</button>

Nota: El botón de "Ejecutar Código" para este ejemplo usará `alert()`. En un entorno real, preferirías un mensaje en la consola o un modal personalizado.

2. JavaScript Interno

Dentro de la etiqueta <script> en el mismo archivo HTML.

Sintaxis del Código:

<!-- En la sección <head> o al final de <body> --> <script> console.log("¡Hola desde JavaScript Interno!"); </script>

3. JavaScript Externo

En un archivo .js separado, enlazado con la etiqueta <script src="...">.

Sintaxis del Código:

<!-- En tu archivo HTML --> <script src="mi_script.js"></script> // Contenido de mi_script.js // console.log("¡Hola desde JavaScript Externo!");

Asignación de Tareas

Tarea: Creación de un script JavaScript con variables y operadores básicos.

Desarrollar un archivo JavaScript (.js) que realice las siguientes operaciones:

  1. Declarar variables usando var, let y const para almacenar diferentes tipos de datos (número, cadena de texto, booleano).
  2. Realizar al menos tres operaciones aritméticas con números (suma, resta, multiplicación, división, módulo).
  3. Utilizar al menos dos operadores de asignación (ej. +=, -=).
  4. Implementar al menos dos comparaciones (ej. >, ===).
  5. Usar al menos un operador lógico (&& o ||).
  6. Mostrar los resultados de todas las operaciones en la consola del navegador (usando console.log()).
