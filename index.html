_
▢
x

¿Qué es un Sistema Operativo? (SO)

Software fundamental que controla el hardware de una computadora y proporciona servicios a los programas de aplicación.

  • Intermediario entre usuario y hardware.
  • Administra todos los recursos del sistema.
Gestión de Procesos
_
▢
x

Gestión de Procesos

El SO asigna tiempo de CPU a cada programa en ejecución (proceso).

  • Permite la multitarea.
  • Inicia, detiene, reanuda y sincroniza procesos.
Gestión de Memoria
_
▢
x

Gestión de Memoria

Administra la memoria principal (RAM) de la computadora.

  • Optimiza el uso de la RAM.
  • Paginación: Divide memoria en bloques fijos.
  • Memoria Virtual: Usa disco duro como RAM adicional.
Gestión de Almacenamiento de Archivos
_
▢
x

Gestión de Almacenamiento de Archivos

Organiza, almacena, recupera, protege y elimina archivos y directorios.

  • Sistema de Archivos: Estructura lógica para datos.
  • Directorios/Carpetas: Agrupan archivos.
Gestión de Entrada/Salida (I/O)
_
▢
x

Gestión de Entrada/Salida (I/O)

Controla la comunicación entre CPU y dispositivos periféricos (teclado, monitor, etc.).

  • Permite interacción usuario-máquina.
  • Mediante Controladores de Dispositivo (Drivers).
Interfaz de Usuario
_
▢
x

Interfaz de Usuario

El medio a través del cual el usuario interactúa con la computadora.

  • CLI (Línea de Comandos): Comandos de texto (ej. Bash, CMD).
  • GUI (Gráfica): Ventanas, iconos, menús (ej. Windows, macOS).
Seguridad y Protección
_
▢
x

Seguridad y Protección

El SO protege los recursos del sistema contra accesos no autorizados y software malicioso.

  • Gestión de permisos de usuario.
  • Autenticación (contraseñas).
  • Cortafuegos básicos y actualizaciones de seguridad.

Sistemas Operativos Móviles

Diseñados para smartphones y tabletas.

Asignación de Tareas

Tarea: Investigación y Presentación de SO

Los estudiantes deberán investigar un sistema operativo de su elección (que no sea Windows o macOS), identificar sus características principales, ventajas, desventajas y ejemplos de uso. La investigación debe culminar en una breve presentación (digital o escrita) que resuma estos puntos clave.

Ponderación: TC 5%.
